# blood-loss-calculator
# 出血量计算工具

一个用于临床出血量评估与计算的Web工具，提供六种不同的计算方法。

## 功能特点

- **休克指数法**：通过心率和收缩压计算休克指数，评估失血量
- **纱布估算法**：通过纱布/敷料数量估算出血量
- **面积法**：通过浸血面积估算出血量
- **称重法**：通过敷料重量变化计算出血量（手术室常用）
- **容积法**：通过吸引瓶液体总量减去原有水量计算出血量
- **血红蛋白法**：通过Hct或Hb变化估算出血量

## 使用方法

1. 下载 `index.html` 文件
2. 在浏览器中打开该文件
3. 根据需要选择计算方法并输入相应参数
4. 点击计算按钮获取结果

## 注意事项

- 本工具计算结果仅供医学专业人士参考，不能替代临床判断
- 任何疑似大出血的患者都必须立即寻求专业医疗救治
- 计算方法基于临床常规指南，实际应用需结合患者具体情况

## 技术特点

- 纯HTML/CSS/JavaScript实现，无需额外依赖
- 响应式设计，支持手机和电脑访问
- 数据本地计算，不涉及隐私数据传输
